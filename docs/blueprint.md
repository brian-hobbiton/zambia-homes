# **App Name**: Zambia Homes

## Core Features:

- User Authentication and Roles: Secure authentication system with distinct roles for tenants, landlords, and admins, ensuring role-based access control.
- Property Listing Management: Landlords can add property details, including title, description, amenities, rent, deposit, and images.  Listings are submitted for admin approval.
- Tenant Application and Lease Management: Tenants can search for properties, view property details, apply to rent, and sign digital rental contracts.
- Automated Contract Generation: Automatically generate rental contract PDFs pre-filled with tenant and landlord info, rent, deposit, and start date.
- Secure Payment Processing: Facilitate rent and deposit payments through integration with a Zambian payment gateway (e.g., Flutterwave or Paystack), storing payment logs and enabling payouts to landlords via escrow.
- In-App Messaging: Enable direct communication between tenants and landlords via an in-app chat feature, facilitating discussions and issue resolution.
- AI-Powered Listing Summarization: Use an AI tool to automatically summarize property listings, highlighting key features and benefits to potential tenants, and creating an improved search and discovery experience.

## Style Guidelines:

- Primary color: Deep teal (#008080) to evoke trust, stability, and connection to the region's natural resources.
- Background color: Light teal (#E0F8F8) to provide a calming and unobtrusive backdrop.
- Accent color: Warm yellow (#FFD700) to represent Zambia's abundant sunshine and add a touch of vibrancy to key elements.
- Body font: 'PT Sans', a humanist sans-serif for readability.
- Headline font: 'Space Grotesk', a proportional sans-serif with a modern feel, used in headlines to establish visual interest. 
- Use clean, modern icons representing property features and actions.
- Maintain a clean and intuitive layout with clear information hierarchy.
- Use subtle animations for loading states and transitions to enhance user experience.
